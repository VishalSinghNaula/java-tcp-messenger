# Java TCP Messenger

A lightweight, console-based chat application built entirely with core Java Networking. 

I set up this project to get hands-on experience with low-level network communication, specifically focusing on TCP/IP protocols, Java Sockets, and Input/Output streams. It establishes a reliable, two-way communication channel between a local server and a client.

## ✨ Features
* **Real-time Communication:** Instant two-way messaging between the server and the client.
* **Core Java Implementation:** Built strictly using `java.net` and `java.io` packages without relying on any external libraries or frameworks.
* **Stream Handling:** Efficient reading and writing of data using `BufferedReader` and `PrintWriter`.
